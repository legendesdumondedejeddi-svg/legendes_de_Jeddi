# Le Monde de Jeddi
🚀 Description du projet

Ce projet est une application web Flask proposant une interface multilingue. Le site détecte ou permet de choisir la langue préférée de l’utilisateur (français, anglais, et toute autre langue que tu voudras lui faire avaler). Le tout est élégamment structuré, facilement extensible, et paramétré pour fonctionner de manière fluide sur Render.

Objectif :
Te fournir une base solide pour créer un site web moderne, international, rapide à déployer, et simple à modifier. Oui, le bonheur existe.
