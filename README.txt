⚠️ STRUCTURE VERROUILLÉE — NE PAS MODIFIER ⚠️

1. NE JAMAIS modifier :
   - css/style.css
   - js/main.js
   - la structure du menu dans les fichiers HTML

2. POUR AJOUTER UNE LÉGENDE :
   - ouvrir legendes.html
   - copier le bloc modèle
   - changer le titre, le texte et l’audio
   - NE PAS toucher aux balises existantes

3. POUR AJOUTER UN AUDIO :
   - placer le fichier .mp3 dans /audio/
   - utiliser exactement le même nom dans legendes.html

4. POUR LES TRADUCTIONS :
   - modifier uniquement les fichiers translations/*.json
   - ne jamais changer les clés

5. Les images doivent rester dans /images/

Si quelque chose casse :
➡️ tu as touché un fichier verrouillé.
